Projects

Obstacle Detector Cum Vacuum Detector:

The Vacuum part was mounted on this bsae car.This vacuum was fitted with motor and a switching system which turns on and off as per the distance of the obstacle.

Seven Segmment Display Using Logic Gates

This was a part of curriculum project of Digital System Lab. In this project the 7-segment display was lighted according to the given input without using the available inbuilt IC for it. AND, NOT, NOR, etc gates were used to implement this project on bread-board and was powered with a 9V battery and a voltage regulator was used along with various resistors for proper functioning

RFID bases Home Security System

This was a part of curriculum project of Microprocessor Lab.It was basically a door unlocking system using electronic key. RFID cards were used as a key and a RFID sensor as a lock. Each valid RFID information was stored in the RFID sensor which autheticates the user. If the RFID card's data matches with the feeded data then it sends positive signals other wise it blows up alarm to alert.

Audio Amplifier using Op-Amp

This was a part of curriculum project of Linear Integrated Circuit Lab. In this project audio was amplified using Op-Amp , resistors, capacitor and transistor.The basic idea was to amplify the input signal without using costly equipment as used in other amplifiers

Water Level Indicator

This project was a part of curriculum project of Circuit lab. In this project the circuit design was so designed that at each particular level of water in the tank it completes a conducting path and that turns on the particular LED indicating the level of water in the tank
